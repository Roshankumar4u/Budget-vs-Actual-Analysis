# Budget vs Actual Variance Analysis

## Objective
To compare planned budgets with actual expenses and identify financial deviations.

## Tools Used
- Microsoft Excel

## Dataset
Monthly department-wise budgeted and actual expenditure data.

## Key Analysis
- Variance and variance percentage calculation
- Identification of over-budget and under-budget areas
- Expense control insights

## Outcome
Provided structured variance reports to support cost control and financial compliance.
